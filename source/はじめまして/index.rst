この勉強会について
------------------

**こんな方を歓迎します**

- プログラミングがはじめての方
- Pythonに興味はあるけど何から始めればいいかわからない方
- 一緒に学ぶ仲間がほしい方

**勉強会の目的**

この勉強会は、プログラミング未経験の方でも無理なく Python を学べる場を目指しています。

- 毎月少しずつ積み上げて、Pythonの基礎を身につけること
- 自分の仕事に活かせるスキルを身につけること
- AIエージェント開発やクラウド活用にもつなげていくこと
- 実際に手を動かして、楽しみながら学ぶこと

**こんな風に進めます**

- 自分の作業を持ち込んで進めていただいても構いません。
- この勉強会には非エンジニア・エンジニアの方両方が参加されています。
- Makinoは先生ではないので、参加者の皆さんと一緒に学んでいけたらうれしいです。
- 分からないところや気付いたことがあれば、ぜひ教えてください。
- 参加者の皆さんが学びやすい環境を作ることを心がけます。
- 参加者同士で助け合いながら、楽しく学んでいきましょう！
